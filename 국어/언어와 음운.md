# 언어와 음운
## 언어의 특징
  1. 언어의 기호성 >> 언어는 어떠한 의미(내용)를 일정한 음성(형식)으로 나타내는 기호이다.
  2. 언어의 자의성 >> 언어의 의미와 음성은 필연적이지 않다.
    - 예: 나무 = き = tree
  3. 언어의 규칙성 >> 언어에는 일정한 규칙이 있다.
    - 예: 나는 어제 학교에 갔다 (O), 나는 어제 학교에 가겠다 (X)
  4. 언어의 분절성 >> 언어는 연속적으로 이루어진 세계를 불연속적으로 끊어서 표현
    - 예: 
      - 의미적 분절: 무지개 빛깔 = 빨강/주황/노랑/초록/파랑/남색/보라
      - 음운적 분절: 감 = ㄱ/ㅏ/ㅁ
  5. 언어의 창조성 >> 무수히 많은 단어와 문장을 만들 수 있다.
  6. 언어의 사회성 >> 언어는 같은 언어를 사용하는 사람들 사이의 사회적 약속
    - 예: '책상'을 마음대로 '양탄자'로 바꾸어 사용할 수 없다.
  7. 언어의 역사성 >> 시간의 흐름에 따라 언어가 변한다. 

## 음운
  - 음성 >> 인간의 발음 기관을 통해 나오는 물리적인 소리로 분절적 소리이다.
  - 음운 >> 말의 뜻을 구별해 주는 가장 작은 소라의 단위
    - 음소 >> 자음, 모음, 반모음
    - 운소 >> 장단, 강세 등

## 음운 현상
  - 음운이 바뀌는 현상
### 교체
  - 음절의 끝소리 규칙 >> 'ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅇ'외의 자음이 음절 끝에 오면 'ㄱ, ㄷ, ㅂ'중 하나로 바뀌는 현상
  - 비음화 >> 'ㄱ, ㄷ, ㅂ'이 비음(ㅁ, ㄴ)앞에서 비음으로 바뀌는 현상
  - 유음화 >> 'ㄹ'과 인접한 'ㄴ'이 'ㄹ'로 바뀌는 현상
  - 구개음화 >> 'ㄷ, ㅌ'이 'ㅣ'모음과 함께 쓰이면 구개음인 'ㅈ, ㅊ'으로 바뀌는 현상
  - 된소리되기 >> 두 개의 안울림소리가 서로 만나 뒤의 소리가 된소리로 발음되는 현상
### 탈락
  - 자음군 단순화 >> 음절 말에서 두 개의 자음 가운데 하나가 탈락하는 현상
  - 'ㄹ'탈락 >> 파생어나 합성어가 만들어지면서 'ㄹ'이 탈락하는 현상, 어간 말 자음 'ㄹ'이 어미의 첫 자음 'ㄴ, ㅂ, ㅅ'및'-(으)오. -(으)ㄹ'앞에서 탈락하는 현상
  - 'ㅎ'탈락 >> ㅎ을 끝소리로 가진 어간이 모음으로 시작하는 어미를 만날 때 'ㅎ'이 탈락하는 현상
  - 모음 탈락 >> 어간 말의 'ㅡ'가 '아/어'로 시작하는 어미 앞에서 탈락하는 현상, 어간 말 모음과 어미 초 모음이 동일할 때 동일 모음 탈락
### 첨가
  - 'ㄴ'첨가 >> 합성어 또는 파생어에서, 앞말의 끝이 자음이고, 뒷말이'ㅣ'나 반모음'j'로 시작하는 경우 'ㄴ'소리가 첨가되는 현상
  - 반모음 'j'첨가 >> 모음으로 끝나는 형태소 뒤에 단모음으로 시작하는 형태소가 올 때 반모음이 첨가되는 현상
### 축약
  - 거센소리되기 >> 'ㅎ'+'ㄱ, ㄷ, ㅂ, ㅈ' → 'ㅋ, ㅌ, ㅍ, ㅊ'
  - 모음 축약 >> 하나의 모음과 하나의 모음이 결합하여 다른 하나의 모음으로 바뀌는 현상
